# Full-Stack Stock Trading Platform (Inspired by Zerodha)

A full-stack stock trading platform inspired by Zerodha, built using the MERN stack.

## 🚀 Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB
- JWT Authentication

## ✨ Features
- User Registration & Login
- Secure JWT Authentication
- Dashboard
- Portfolio Management
- Holdings & Positions
- Responsive UI
- REST API Integration

## 📂 Project Structure

frontend/
backend/
dashboard/


## ⚙️ Getting Started

### Frontend
bash
cd frontend
npm install
npm start


### Backend
bash
cd backend
npm install
npm start


### Dashboard
bash
cd dashboard
npm install
npm start
